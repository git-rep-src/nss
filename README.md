# Nss - Ncurses search sploit

[Exploit-DB's Offensive Security](https://github.com/offensive-security/exploit-database) Ncurses search tool.

## INSTALLATION

```shell
ncurses  >= 6.0
libxml++ >= 3.0
```
```shell
make
make install
```
## CONFIGURATION

~/.nss

```shell
git=/path/exploit-database/
copy=/path/my-exploits/
editor=editor
```
